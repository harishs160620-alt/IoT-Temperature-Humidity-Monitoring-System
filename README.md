# IoT Temperature & Humidity Monitoring System

## Project Overview

This project simulates an IoT-based Temperature and Humidity Monitoring System.

## Objective

To monitor environmental conditions using temperature and humidity sensors and generate alerts when values exceed predefined thresholds.

## Components

- Temperature Sensor (Simulated)
- Humidity Sensor (Simulated)
- Python
- Visual Studio Code

## Working

1. Sensor readings are generated.
2. Temperature and humidity values are displayed.
3. If temperature exceeds 30°C:
   - High Temperature Alert is generated.
4. If humidity exceeds 70%:
   - High Humidity Alert is generated.

## Applications

- Smart Agriculture
- Weather Monitoring
- Smart Buildings
- Industrial Automation

## Output

Temperature : 35°C
Humidity : 75%

Temperature Alert : HIGH
Humidity Alert : HIGH
